## 📋 GitHub Pages 설정 방법

1. GitHub 저장소 페이지에서 **Settings** 탭을 클릭합니다
2. 왼쪽 메뉴에서 **Pages**를 클릭합니다
3. **Source** 섹션에서:
   - "Deploy from a branch" 선택
   - Branch: **main** 선택
   - Folder: **/ (root)** 선택
4. **Save** 버튼을 클릭합니다

설정 완료 후 5-10분 정도 기다리면 웹사이트가 활성화됩니다.
